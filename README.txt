JPadder
-------
Map controller events to keybord/mouse events
Runs on Windows, Linux and Mac


Project setup
-------------
some maven dependencies aren't available with default configuration.
add folowing to local repository:
	<dependency>
		<!-- compile from https://github.com/brunothg/GameEngine.git --!>
		<groupId>de.bno</groupId>
		<artifactId>de.bno.gameengine</artifactId>
		<version>1.2.2</version>
	</dependency>
	<dependency>
		<!-- compile from https://github.com/brunothg/Swing2.git --!>
		<groupId>de.bno</groupId>
		<artifactId>de.bno.swing2</artifactId>
		<version>0.0.5</version>
	</dependency>
